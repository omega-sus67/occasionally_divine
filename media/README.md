## Clips (`clips/`)

Each moment is provided twice:
- **`.gif`** — drop straight into the GitHub README (autoplays inline, no click).
- **`.mp4`** — upload to LinkedIn as native video (crisper, smaller, better reach than GIF).

| Clip | Shows |
|------|-------|
| `season-transition` | Year/season transition card fading in |
| `elders-debate` | Council members debating, then the AI proposal resolving |
| `oracle-of-the-archives` | Asking the Oracle a question and its answer streaming in |

Upload the `.mp4` files directly as video. The `elders-debate` and `oracle` clips are
